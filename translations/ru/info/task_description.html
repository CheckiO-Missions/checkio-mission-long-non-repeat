Существует четыре миссии связанные с анализом частей строк. Все они были созданы за один день и не требуют более одного дня для своего решения. Эти миссии можно с легкостью преодолеть посредством простого перебора символов. Но, возможно, есть вариант получше? (У Вас может еще не быть доступа ко всем этим миссиям, но по мере открытия островов на карте он будет предоставлен)

Это вторая миссия, которая по своей сути схожа с первой миссией серии, но требует другого направления поиска. Здесь Вам необходимо найти первую самую длинную подстроку состоящую исключительно из неповторяющихся букв. Например, в строке "abca" мы имеем две подстроки с неповторяющимися буквами "abc" и "bca", но нам нужна первая подстрока, поэтому ответом будет "abc".

Входные данные: Строка.

Выходные данные: Строка.

nonRepeat('aaaaa') == 'a'
nonRepeat('abdjwawk') == 'abdjw'
nonRepeat('abcabcffab') == 'abcf'
{% else %}
non_repeat('aaaaa') == 'a'
non_repeat('abdjwawk') == 'abdjw'
non_repeat('abcabcffab') == 'abcf'
{% endif %}
